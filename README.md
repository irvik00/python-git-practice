# python-git-practice
## Репозиторий для практики работы с Git

Этот репозиторий создан в рамках учебного задания по программированию на Python для отработки навыков работы с системой контроля версий Git.

### Содержание

* `calculator.py` - простой модуль калькулятора с базовыми арифметическими функциями (сложение, вычитание, умножение, деление).
